RobloxClasses


This module is used to get methods, properties or events from Roblox object classes. It can also be used to know various information about any class, function, property or event, by checking the presence of certain tags (i.e. hidden, readonly, deprecated). For more informations, visit http://wiki.roblox.com/index.php?title=API:Class_reference/Dump/Help.


Methods


boolean ClassHasTag(string className, string tag) Returns if a class has a tag
table GetClassEvents(string className) Returns the events from the class with the inherited ones
table GetClassFunctions(string className) Returns the functions from the class with the inherited ones
table GetClassMembers(string className, string memberType)
table GetClassProperties(string className) Returns the properties from the class with the inherited ones
void MemberHasTag(string className, string memberType, string name, string tag) Returns if a certain field has a tag
