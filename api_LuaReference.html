LuaReference


This class is meant to make it easy to get informations about the lua built-in functions (tostring, tonumber, the math and string libraries).


Methods


table GetArguments() Returns the arguments of the function. If the reference is not to a function, it returns an empty list.
void GetReferenceInfo() Updates the members of the object.
table GetReturnValues() Returns the return values of the function. If the reference is not to a function, it returns an empty list.
string GetTitle()
string GetValueType()
boolean IsEvent() Returns if the reference is an event. Always false for a LuaReference.
boolean IsFunction() Returns if the reference is a function
boolean IsValue() Returns if the reference is a value.
table Load(table data) Creates a new object with the given table.
table New(string referenceName) Create a reference to a built-in function in lua.
table Serialize() Serializes all the object data in a table to be reloaded using the Load method.
